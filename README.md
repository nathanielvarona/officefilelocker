officefilelocker
================

Office File Locker (Jython CLI and Apache POI)

### Dependencies
 - [Jython CLI](http://www.jython.org/)
 - [Apache POI](http://poi.apache.org/)

### Usage

Locking an Office File:
```bash
$ jython officefilelocker.py -u <username> -p <password> -i <inputfile> -o <outputfile>
```
